## Software & Scrapers
Tools that "claw" data from websites, APIs, or log sources.

* [clawpack/clawpack](https://github.com/clawpack/clawpack) - A powerful Python-based finite-volume method library for hyperbolic PDEs (Python).
* [leeroyduff/ClawCrawler](https://github.com/leeroyduff/ClawCrawler) - A modular web scraping and extraction framework for structured data collection (Python).
* [matthai/Claw-Data](https://github.com/matthai/Claw-Data) - Data ingestion and transformation utilities for product and sales data clawing (Node.js).

## Robotics & Hardware
Open-source mechanical claws, robotic arm controllers, and 3D printable hardware.

* [Pololu/RoboClaw](https://github.com/pololu/roboclaw) - Motor controller firmware and Python/C# control tools for robotic claws (C/C++, Python).
* [udacity/robotics-claw-kit](https://github.com/udacity/robotics-claw-kit) - Building and controlling a claw arm with Raspberry Pi and servos (Python).
* [jbrophy/hardware-claw](https://github.com/jbrophy/hardware-claw) - 3D-printable claw mechanism with stepper motor drive and Arduino code (C++).

## Games & Emulators
Claw machine games, physics-based claw simulators, and interactive claw-themed experiences.

* [appleple/ClawGame](https://github.com/appleple/ClawGame) - Unity-based claw machine game with procedural item physics (C#).
* [sammgwhe/retro-claw](https://github.com/sammgwhe/retro-claw) - HTML5/Canvas arcade-style claw game emulator for browsers (JavaScript).
* [openclaw/stage-claw](https://github.com/openclaw/stage-claw) - Game engine extension module for claw dynamics and input control (C++).

## Libraries & Frameworks
Re-usable libraries, SDKs, and toolkits that use the name "claw".

* [LennartHennigs/claw](https://github.com/LennartHennigs/claw) - A fast C++ networking library with streaming and parsing utilities.
* [n8io/claw.js](https://github.com/n8io/claw.js) - JavaScript CLI framework for building fast command line tools (Node.js).
* [clawlabs/react-claw](https://github.com/clawlabs/react-claw) - React components and animation utilities for interactive claw UIs.

## Community

### How to contribute
1. Fork this repo.
2. Add a new bullet under the correct category.
3. Keep entries sorted alphabetically by repo name.
4. Add one sentence and include language/tech tags.
5. Create a PR and link any supporting demos or docs.
