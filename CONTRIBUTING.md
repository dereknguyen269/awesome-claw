# Contribution Guidelines

Please ensure your pull request adheres to the following guidelines:

- Search previous suggestions before making a new one, as yours may be a duplicate.
- The repository being added must be related to **OpenClaw** (skills, forks, alternatives, dashboards, deployment, memory, etc.).
- The repository should have a clear README and be actively maintained or functionally complete.
- Make an individual pull request for each suggestion.
- Keep descriptions short, simple, and objective.
- Ensure the link is functional.
- Check your spelling and grammar.

### Formatting
Please use the following format when adding a new project to the list:

`* [owner/repo](https://github.com/link) - A short, one-sentence description. (Main Language)`

Thank you for your suggestions!
